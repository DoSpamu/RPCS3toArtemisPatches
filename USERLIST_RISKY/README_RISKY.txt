===========================================

This folder is a copy of USERLIST with additional RPCS3 patches
that were added DESPITE a game version mismatch.

WHAT THIS MEANS:
- The regular USERLIST folder contains only patches matched to the exact game
  version (e.g. a v01.00 patch was added only to .ncl files for v01.00).
- This folder (USERLIST_RISKY) ALSO includes patches from other game versions.

THE RISK:
- A patch written for game version 01.09 may use DIFFERENT memory addresses
  than version 01.00.
- If the version does not match, the patch may:
    * do nothing
    * freeze the game
    * corrupt save data (rare, but possible)
- Worst case: game crash, return to XMB.

HOW TO IDENTIFY RISKY PATCHES:
All patches added with a mismatched version include the version number in their
name, for example:
    "Unlock FPS v01.04 (RPCS3)"   <- patch written for v01.04
Safe patches (version matched) have no version number in the name:
    "Unlock FPS (RPCS3)"          <- version matched

WHAT TO DO IF A PATCH DOES NOT WORK:
1. Disable the patch in Artemis
2. Update the game to the version shown in the patch name
3. Or use the USERLIST folder (safe) instead of USERLIST_RISKY
