# ⚠️ USERLIST_RISKY — Use at Your Own Risk

This folder is a copy of `USERLIST/` with **additional RPCS3 patches that were added despite a game version mismatch.**

## What this means

- **`USERLIST/`** (safe) — contains only patches matched to the exact game version. A v01.00 patch is added only to `.ncl` files for game version v01.00.
- **`USERLIST_RISKY/`** (this folder) — also includes patches from **other game versions**, where no exact version match exists.

## The risk

A patch written for game version `01.09` may use **different memory addresses** than version `01.00`.
If the version does not match, the patch may:

- do nothing
- freeze the game
- corrupt save data *(rare, but possible)*
- worst case: game crash, return to XMB

## How to identify risky patches

Patches added with a mismatched version **include the version number** in their name:

```
Unlock FPS v01.04 (RPCS3)   ← written for v01.04, risky if your game is a different version
Unlock FPS (RPCS3)           ← version matched, safe
```

## What to do if a patch does not work

1. Disable the patch in Artemis
2. Update the game to the version shown in the patch name
3. Or use the `USERLIST/` folder (safe) instead
