How to use
----------
1. Connect to PS3 via FTP
2. Copy the .ncl file to: hdd0/game/ARTZ00001/USRDIR/USERLIST/
3. Open Artemis, select game, enable the FPS/FOV patch
4. Launch game, press PS + Start to attach

Want more patches? Use ../USERLIST/ (all patches including RPCS3-sourced ones).
